# Teams But Actually Good — Website

Website for [Teams but (actually) good](https://github.com/LeonimusTTV/teams-but-actually-good)

## Dev

```bash
bun install
bun run dev
```

## Build & Preview

```bash
bun run build
bun run preview
```

## Production

```bash
bun run start
```

Serves the built output via `server.mjs`.
